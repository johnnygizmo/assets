About This Library

This is an online Blender asset repository hosted at the URL below. It contains curated asset packs including furniture, electronics, housewares, lamps, planters, shelves, and more — ready to drag and drop directly into your Blender scenes via the Asset Browser.

Add the URL to Blender's Asset Library preferences and all assets become immediately available without downloading anything manually.

https://johnnygizmo.github.io/assets/

How to Add This Library in Blender

Requires Blender 4.3 or later. Tested with Blender 5.2.

  1. Open Edit → Preferences (or press Ctrl + ,).
  2. Select the File Paths section from the left sidebar, then scroll down to Asset Libraries.
  3. Click the + (Add) button in the Asset Libraries list to add a new entry.
  4. In the new entry, change the type from Local to Online Library using the drop-down on the left side of the row.
  5. Paste the URL above into the URL field.
  6. Give the library a name (e.g. JohnnyGizmo Assets) — this is only used inside Blender to identify it.
  7. Set your preferred Import Method (see below), then close Preferences. The library will appear in the Asset Browser immediately.
Note: Online access must be enabled in Blender. Go to Edit → Preferences → System and ensure Online Access is turned on.

Import Methods

Choose how assets are brought into your blend-file when you use them from the Asset Browser.

Method Description
Append Creates a fully independent copy of the asset and all its dependencies (meshes, materials, etc.). Every import creates new data. No ongoing connection to this library.
Append (Reuse Data)Recommended Like Append, but Blender remembers previously imported assets and reuses compatible data on subsequent imports — avoiding duplicates for things like shared materials or meshes.
Pack Imports as linked data and immediately packs it into the blend-file, creating a self-contained file. Useful for archiving or sharing projects.
Note: Link is not supported for online asset libraries. Use Append, Append (Reuse Data), or Pack instead.
